## Gulp tasks
### Deploy
`$ gulp deploy` creates a production ready version inside a folder **_DATE(YYMMDD)-PROJECT-TIME(HHMM)**

		ex: _140827-my-project-1139

This "production version" excludes the following files/folders:
* _dev/\*.\*
* node_modules/\*.\*
* checklist.todo
* gulpfile.js
* package.json
